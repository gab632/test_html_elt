HTML Media Element (audio and video)

The <video> HTML element embeds a media player which supports video playback into the document.

Note: The <video> HTML element can be used to play audio files, too

source

The <audio> element is used to embed sound content in documents.
It may contain one or more audio sources, represented using the src attribute or the <source> element:

Note: The <video> HTML element can be used to play audio files, too

source

Several HTML attributes are avaialble for the <video> and <audio> elements

For example, "loop" and "autoplay" (alongside "muted")

Note: Some browsers require the "muted" attribute to have the video start automatically.

source
